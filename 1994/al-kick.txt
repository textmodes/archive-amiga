  | Why do I live? Have you ever asked yourself that question? If not, may- |
  | be it's about time you do! Some pessimistic persons might say that we   |
  | live only to die! I think not! We all probably have different answers   |
  | to this question but I think all people have at least one thing in com- |
  | mon: We all want to be happy! But we all are different and therefore    |
  | different things make us happy! I, for example, am extremely happy      |
  | whenever I'm skateboarding because that's the best thing I know! Some-  |
  | one else might get happy by painting, surfing, making music, biking,    |
  | coding or whatever, we all have our moments of joy and we can't go on   |
  | without them! What I'm trying to say is that we all are different and   |
  | we can't expect everyone to like everything we like! I'm not very fond  |
  | of soccer for example, but I don't dis soccer players or talk shit      |
  | about it just because of that! It's their thing and if it makes them    |
  | happy then it's just fine with me! Why should I ruin someone elses      |
  | happiness? That would make me just as bad as the people dissin' me      |
  | because I'm skateboarding, and I can't expect anyone to respect me for  |
  | what I am if I can't respect them for what they are! That's what it's   |
  | all about: RESPECT! You can't like everything everyone else is doing    |
  | but you should at least be able to respect it! What good does it make   |
  | to hate? Hate only breeds hate! Wouldn't everything be so much better   |
  | if we wouldn't prejudge everyone and just like them for what they are?  |
  | I think so! Think about it! Tack, and may the edge be with you!         |
