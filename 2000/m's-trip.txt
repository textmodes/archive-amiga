 Hello and welcome to my last colly under MO'SOUL label!   |             |
 Yes you heard it right, this colly was originally going   |             |
 to be released in the last co-op collection from the whole|             |
 MO'SOUL gang, but as it seems now most members have become|             |
 inactive or left for other groups, so i decided to release|             |
 this one for myself!                                      |             |
 This is no "i leave MO'SOUL" collection, no! This is just |             |
 my way to salute this great group that i have had the     |             |
 opportunity to be part of for nearly 4 years!! I dont     |             |
 really know what the deal with MO'SOUL is really, are we  |             |
 dead are we alive? But for me there are no other options  |             |
 to when it comes to join other groups, there are no group |             |
 on the asciiscene that has even come close to the quality |             |
 releases that MO'SOUL has been pulling out the past 4-5   |          /  |
 years except the Arclite guys! (Great new colly H2o!)     |        ///  |
 There wont be anymore collections from me released under  |       .//   |
 MO'SOUL in the future, but still the cDr/m's tag will     |       /     |
 remain the same and stick to all my new coming artworks   |             |
 as i still feel like i belong to this group, eventhought  |             |
 the releases will be pulled out under independent label!  |             |
                                                           |             |
 This collection is a little "remix" thing from all of my  |             |
 collections released under MO'SOUL label, just a lookback |             |
 on a few logos from each colly ive released during the    |             |
 4 years(since 96') ive been in this group, in total five  |             |
 collections. I grabbed the two nicest logos from each     |             |
 colly that *I* liked the best when i look at them now,    |             |
 check em out! Maybe you like em aswell..                  |             |
 This is also my way to show you how lazy i am doing logos,|             |
 so instead of doing lots of new logos i grab logos from my|             |
 old collies and recycle them! :p The first part in this   |             |
 trip is where i take you back in time to the days when i  |             |
 began drawing ascii's and the things that got me hooked to|             |
 this artform! In early 94' when i began i was not part of |             |
 the real modem/asciiscene i didnt own any modem and i was | oOo         |
 mailswapping alot with friends from all around the world, |             |
 some guys who i still have contact with! (Ghandy, Exon,   |             |
 Markryder, Devistator, Opel etc..) In those days my main  |             |
 contact with asciiscene was checking out bbs adds from    |             |
 various boards and watching a few collies every now and   |             |
 then. I used to collect cool looking bbs adds in a .txt   |             |
 file and i still got em here, burned on CD along with the |             |
 rest of the collies 2000+ that ive collected during the   |          __ |
 years, it surely brings lots of good memories back!       |         _\/ |
                                                           |             |
 Lets start this colly with a bunch of logos that i have   |             |
 gathered with a few of my favourite bbs adds ever, that   |             |
 made *ME* intrested in the asciibiz and the logos which   |             |
 still means alot to me! In total 9 logos from various     |             |
 known and unknown artist, with a little comment on them   |             |
 from me, just check em out and see what you think!        |             |
 The second part is the stuff i released during the MO'SOUL|             |
 years, i hope you enjoy the trip!                         |             |
 this is my story..                                        |             |
