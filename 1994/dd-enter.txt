  |       hi folks ! this is rapid of dream design speaking .. i know       |
  |     there are lots of other ascii groups out there, so you may ask:     |
  |   why the fuck another one ?! well, everybody can see that the ascii    |
  |  business is going down nowadays .. almost every week new productions   |
  |  of wanna be ascii 'heros' are released, but that isn't art .. i miss   |
  |    the high quality productions of the -real- old school, you know      |
  |  what i'm talking about .. what has happened to all the great masters   |
  |    in the ascii scene ?? come on guys ... we need you! ok, now a new    |
  |        group enters the ascii business .. i am finally back since       |
  |      i stopped releasing ascii-productions a half year ago because      |
  |  i had some silly problems and there were no fun anymore .. but that's  |
  |        history i hope, now it's time to break the limits again !!       |
  |    now i'm back .. back with inspiration .. back with dream design !    |
  |       the dream designers are rapid and pump .. we don't wanna be       |
  |  self-nominated ascii hero's but accept us nevertheless .. thank you !  |
  |                                                                         |
  |                                                                         |
  |               special thanks to [-PumP-] .. we'll make it!              |
  |                                                                         |
