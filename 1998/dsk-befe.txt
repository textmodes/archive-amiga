                   _____ __ HOWTO DO ELITE ASCII LOGOS ____::....::
       -  -- ---- /____//_/                           /___/ ------ --------
                       .
                       ::   YESH, NOMORE YOU NEED TO BEGGAR ON
                       ::   #ASCII FOR THAT ELITE ASCII LOGO
                       ::   YOU NEEDH.
                       ::
                       ::   STEP ONE. FIGURE IF YOU REALLY NEED
                       ::   THE LOGO. THIS IS VERY IMPORTANT,
                       ::   SINCE CREATING A LOGO REQUIRES SOME
                       ::   AMOUNT OF GOOD QUALITY WEED TO
                       ::   GAIN A HIGHER STATE OF CREATIVITY.
                       ::
                       ::   STEP TWO. FIGURE WHAT THE LOGO SHOULD
                       ::   SAY. EG. "SMOKE".
                       ::
                       ::   STEP THREE. DRAW AN OUTLINE BOX FOR THE
                       ::   LOGO. APPROX 10 CHARS WIDE FOR EACH
                       ::   CHAR IN THE LOGO. EG. "SMOKE" IS
                       ::   FIVE CHARS, THEREFORE THE BOX SHOULD BE
                       ::   MINIMUM 50 CHARS WIDE (SINCE 5*10=(2^0+2^2)*
                       ::   (2^1+2^3)=(2^0*2^1)+(2^0*2^3)+(2^2*2^1)+
                       ::   (2^2*2^3)=2^3+2^3+2^1+2^5=8+8+2+32=50
                       ::   (DONT WE LUB BINARY CALCS?))
