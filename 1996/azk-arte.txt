      If your board has something like UPLOAD RULES and one of the rules
       says that every ascii collection uploaded receive bytes * 2 then
             tell to your favourite door-coder to make this one:


         o   After upload it calls this procedure:

                - check if the file has TXT extension
                - check if the file is bigger than 30000 bytes
                - check if the filename has ascii-groups shortage e.g
                        AZK-abcd.TXT  for Azkiness
                        -T-abcde.TXT  for Twisted
                        SOS-abcd.TXT  for Save Our Souls
                  (you can search for list of groups in Ascii Gazette by xCz)
                - give bytes * 2 to uploader
                - send message about given bytes to uploader
                - exits

        o   The End
