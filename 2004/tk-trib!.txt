  ::  This collection is a tribute to the long-past days of Trank!Ascii.  :::
  ::                                                                      :::
  ::  Whodini created Trank as a logical continuation of Remorse, but in  :::
  ::  time we surpassed our own point of origin and achieved far more in  :::
  ::  terms of fostering the development of new artistic expressions and  :::
  ::  styles, and certainly did more for creating an international ascii  :::
  ::  scene than any other group had until the Remorse reformation.  The  :::
  ::  popularity of "new school" ascii, collections in the pc scene, and  :::
  ::  the use of amiga fonts by non-amigers are all debts owed to Trank.  :::
  ::                                                                      :::
  ::  Trank was an exicting time in the scene, as we broke down barriers  :::
  ::  between platforms and challenged everyone to achieve more in their  :::
  ::  art.  We, as Whodini was fond of saying, ate groups for breakfast.  :::
  ::  We also caused the creation of a few groups (NoName, AsCiiD!) with  :::
  ::  our strict "invite only" policy on new recruits.  We sparked a war  :::
  ::  or two (the big one with Mo'Soul most notibly), and made some good  :::
  ::  friends along the way too (Save Our Souls and Style stick out most  :::
  ::  prominently in my memory).  Like I said, it was good scene times..  :::
  ::                                                                      :::
  ::  Remorse's inner-clique, the 1981, is really a continuation of what  :::
  ::  we started with the wU!tRANK cLAN: a group within the group; dedi-  :::
  ::  cated to one another and the group above all others. sTYLE.2.dEATH  :::
  ::  wasn't just a slogan, it was a way of life; and it still is..       :::
  ::                                                                      :::
  ::  This is my first collection since 2k1, and my last under the alias  :::
  ::  Hiro Protagonist.  There are a few reasons I've wanted to make the  :::
  ::  change and drop the alias, and most of them are centered around my  :::
  ::  need for creative originality and honesty in my art.  But thinking  :::
  ::  about Trank while making this collection I knew I had to represent  :::
  ::  as Hiro this one last time.  So, uhm, I need some Devin logos now,  :::
  ::  if you could be so kind.. =]                                        :::
  ::                                                                      :::
  ::  My next collection, titled Homeless, is nearly finished and should  :::
  ::  be released in Remorse #55, the ten year anniversary pack, and can  :::
  ::  be found on the usual boards as well (big ups to The Yard).  There  :::
  ::  isn't a lot of variation in this collection, but there is a lot to  :::
  ::  be found in Homeless so I hope you'll forgive me on this one.  I'm  :::
  ::  looking forward to a little feedback on this, and welcome requests  :::
  ::  or general emails as well.  Let me know if you're breathing..       :::
  ::                                                                      :::
  ::  I drew this using True!School's Topaz font and obviously reccomend  :::
  ::  using it to view, but 8Ox5O afs will suffice if it's all you have.  :::
  ::  Also, it's worth mentioning that I draw with white text on a black  :::
  ::  background (I fucking hate notepad) and reccomend you use the same  :::
  ::  settings to view.                                                   :::
  ::                                                                      :::
  ::  Thanks and greetings go out to: Whodini, Omicrom, and Kaleidas for  :::
  ::  their hard work running Trank; all the member artists of Trank for  :::
  ::  their wonderful contributions; Mark Ryder for being an asshole; my  :::
  ::  guest artists for making this collection look prettier, and to you  :::
  ::  for reading and indulging me in my art and need for an audience.    :::
