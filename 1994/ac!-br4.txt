                     | Well, everything start once   |
                     |  upon in time...              |

                     | It was in September, in one   |
                     |  of the coolest years back in |
                     |  1992. At this time I started |
                     |  calling out with my SUPRA    |
                     |  2400BPS modem, which I bought|
                     |  at the present time, and     |
                     |  started calling out on c64!  |
                     | At this time everything was   |
                     |  possible... you could break  |
                     |  nearly o13o-number and I just|
                     |  called 3-4 hours the bbs's in|
                     |  the States... At this time,I |
                     |  actually was a member of the |
                     |  c64-division from TRSI-amiga |
                     |  named RSI (RED SECTOR INC.). |
                     | As we got built up we kicked  |
                     |  really every other major ass |
                     |  so other groups, like G*P,   |
                     |  Legend, Illusion, Dominators |
                     |  and also Deadline had no     |
                     |  chance to beat us in quality |
                     |  and in the quantity of all   |
                     |  releases... we released      |
                     |  nearly 8o% of all games as   |
                     |  the FIRST and also pal/ntsc  |
                     |  fixed them for the Euro's/   |
                     |  American's.                  |
                     | In the first week I called the|
                     |  States, a new bulletin board |  ___ _ __ _
                     |  system opened and got around | /_/\ _ _
                     |  2o-3o users after one week   | \_\/_ ___
                     |  and also nearly 25-3o calls  |
                     |  per day!                     |
                     | You have to know, that it's   |
                     |  not possible to get more than|
                     |  one node at the c64, so it   |
                     |  was quite enough for such an |
                     |  unkown bbs...                |
                     | Nearby I did some bbs's gfx,  |
                     |  which really look better, as |
                     |  the normal c64-charset is    |
                     |  quite good designed and it   |
                     |  was great to design s | ome     |
                     |  colourfull ansi's!           |
                     | So I did all bbS-graphics for |
                     |  the new bbs and also edited  |
                     |  all prompts for the users in |
                     |  colour... after 3 months, I  |
                     |  stopped calling out, 'caused |
                     |  by an student-exchance with  |
                     |  a technical-school in Rich-  |
                     |  mond(England/UK)...          |
                     | After being away from my c64  |
                     |  a lot of things happend in   |
                     |  this time to dEATHLORD(the   |
                     |  sysop of the bbs where I     |
                     |  called! At first he got      |
                     |  busted by aT&T, because his  |
                     |  bbs had an +1-8oo'er, so it's|
                     |  was total-free for all Amis, |
                     |  like here in Germany the o13o|
                      daand he also got probs 'cause |
   ___ _ __ _        |  of hacking pbx's,cc via voice|
  /_/\ _ _           |  and also via computer...     |

                     |  Butch is still sitting in    |
                     |  jail... In 1997 he will get  |
                     |  get out... so "just" 3 years |
                     |                               |
                     | His bbs had the holy coolest  |
                     |  name I ever had heard...     |
                     |                               |
