                               Why 5ive?                                 :::
                                                                        :::
    - I Always Hate  To  Write These  Intros,  If You  Have  Seen My     :::
    Previous  Collections  You  Will  See That Have All Always Hated      :::
    To  Write  These.  But  Why  5ive?  The  First And  Most Obvious       :::
    Reason Is Because  This Is My  Fifth Collection  And  The Second      :::
    Reason  Is  Because  I  Can  Celebrate  5ive  Years   Of  Scene-     :::
    Activity.  I've  Now Been  Trading  More Or  Less  Serious Since      :::
    1993  And  I  Have  Been  Doing  Ascii  Since  Late  1995.  I've     :::
    Released  Four Ascii  Collections  In  1996,  But During  1997 I    :::
    Lost  The Feeling For Ascii.  I've Only Made About 40-50 Asciies     :::
    Since My Last  Collection,  And That's Why 5ive Are Released. Or      :::
    Should  It Have  Been Called  5inal? The  Official Title Of This       :::
    Should  Have  Been "Lost  And Found",  A Name Stratos  Of Contra      :::
    Mend Would  Be  Perfect  For My Collection.  And Another  Thing,     :::
    I'm  Not  Fan In  Any  Other  Way  Of  The Boygroup  Named 5ive.    :::
                                                                         :::
    The Ascii-Scene As We  Know It Is Dead. We Never See Collections      :::
    From Groups Like Arclite,  Epsilon Design,  Save Our Souls, Jedi       :::
    Arts And Style Anymore. I Know There Is Alot Of Other Crews Too,      :::
    But They Was The Most Active Ones During 1996 And 1997. But Why?     :::
    Are Ascii On The  Way To Die With The BBS's?  Is It Because They      :::
    Die That We  Don't Need Cool Ascii For Them Anymore. I Think So,     :::
    Why Bother  To Make  A Logo If No One  Is Going  To  See It. And    :::
    Ascii Look Terrible On The Internet. Too Bad, But That's A Fact.     :::
                                                                          :::
    In This Collection  I've Included About 3O  Logos, All Requested       :::
    During  1997  And  1998 + Some  Of  My Favourites  From Previous      :::
    Collections. I  State That This  Is My Final  Collection, It Is.     :::
    But If  You Need A  Logo,  And You Can Promise  Me That You Will    :::
    Use It  And People Will  See It, I Will Try Make  What You Need.     :::
                                                                          :::
                       eMail Me For Any Requests.                          :::
                                                                          :::
    Enough Talk,  Let's Start The End Of An Era.  Let's Launch 5ive:     :::
                                                                          :::
